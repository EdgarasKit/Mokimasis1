Klasių, ID ir tag'ų selektoriai

Klasės

Norint kreiptis į klasę iš CSS'o - CSS reikia parašyti "." ženklą ir tos klasės vardą. Viena klasė gali būti skirta daugeliui elementų. Vienas elementas gali turėti daug klasių. Klasėse nurodytas vienodas stilius yra prioritizuojamas pagal tai, kelintas eilėje jis yra aprašytas CSS faile. Tėvinio elemento vienodas CSS stilius yra prioritizuojamas senelinio (tėvinio tėvinio) elemento atžvilgiu.

ID

Norint kreiptis į ID iš CSS'o - CSS reikia parašyti "#" ženklą ir to ID vardą. Vienas ID turi būti priskiriamas vienam elementui. Vienas elementas gali turėti tik vieną ID. Tėvinio elemento vienodas CSS stilius yra prioritizuojamas senelinio (tėvinio tėvinio) elemento atžvilgiu.
